# curriculumweb5
Next.js + TypeScript + Tailwind Project

This project has been setted up following this tutorial: https://www.kyrelldixon.com/blog/setup-nextjs-with-tailwindcss-and-typescript

## Getting started

Install node, npm. Then, run:
```bash
npm install
```

## Develop run

```bash
npm run dev
```

## Run server

```bash
npm run build

npm start
```
